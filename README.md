#duckyCar

This repository contains code used on an elegoo robot car kit v3.0

Hardware used:
elegoo Mega 2560 R3
Adafruit L3GD20 Gyro
Pixy Camera
SunFounder PCA9685 16-channel 12-bit PWM Servo Driver

Required libraries not included here:
Wire
Adafruit_PWMServoDriver
Adafruit_L3GD20_U
Adafruit_L3GD20
Adafruit_Sensor
Pixy
MemoryDelegate
SPI
